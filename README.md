Запуск проекта:
1. Перейти в папку с решением, где находится docker-compose
2. В пути проводника ввести cmd, либо powershell
3. Использовать команду "docker-compose up", запустится весь backend
4. Перейти в папку с проектом фронта (my-app)
5. Тоже в пути проводника ввести cmd, либо powershell и запустить через команду npm install, а затем npm run dev.


Примечание:
npm install запустит скрипт для скачивания необходимых библиотек. Для повторного запуска его вводить не нужно
